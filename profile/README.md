## Overview

macOS provides a solid foundation of input device support and keyboard shortcut management, but for power users who want to push the boundaries of what their hardware can do, the built-in capabilities barely scratch the surface. <a href="#">BetterTouchTool</a> extends every aspect of macOS input handling to its theoretical limit, providing a comprehensive customization environment that transforms how every pointing device, keyboard, and gesture input method functions on your Mac. The trackpad gesture system alone supports configurations of extraordinary complexity — any combination of finger count, motion direction, pressure, and tap pattern can be mapped to any system action, application command, or <a href="#">custom automation sequence</a>. Users who invest time in configuring their gesture vocabulary report dramatic improvements in the speed and fluidity of their Mac workflows.

The <a href="#">window management</a> capabilities in BetterTouchTool address one of the most commonly cited productivity pain points in macOS — the absence of a built-in window snapping and tiling system comparable to Windows. BetterTouchTool's drag-to-snap zones appear as visual guides when windows are dragged near screen edges or corners, allowing precise placement into predefined layout positions with a simple gesture. Advanced users can define custom snap zones with specific dimensions and positions, creating personalized <a href="#">workspace layouts</a> that organize their most-used applications into efficient screen arrangements. Keyboard-based window positioning commands provide an alternative workflow for users who prefer not to use the mouse for window management, with every position and size parameter configurable through the preferences interface.

The depth of <a href="#">BetterTouchTool's</a> automation capabilities extends well beyond input remapping into genuinely programmable Mac control. The action sequencing system allows complex multi-step operations to be triggered by any supported input event — opening applications, executing shell scripts, sending keystrokes to specific applications, controlling system settings, and displaying <a href="#">HUD notifications</a> — with conditional logic, delays, and loops that make sophisticated automation workflows practical to implement without coding. The <a href="#">clipboard manager</a> and snippet expansion features add text productivity capabilities that complement the gesture and shortcut system, creating a comprehensive productivity layer on top of macOS. For users who want to extract the maximum capability from their Mac hardware, <a href="#">BetterTouchTool OSX</a> is the single most impactful customization tool available.

## Key Features

- Unlimited custom <a href="#">trackpad gestures</a> with full control over finger count and motion
- Advanced <a href="#">keyboard shortcut</a> remapping and macro creation for any application
- Powerful <a href="#">window snapping</a> and management with drag-to-snap zones
- Complete <a href="#">Touch Bar customization</a> with widgets, buttons, and app-specific layouts
- Custom actions for <a href="#">Magic Mouse, scrolling</a>, and modifier key combinations
- Clipboard manager with <a href="#">history and snippet</a> expansion capabilities
- Advanced <a href="#">trigger types</a> including time-based, app-focused, and remote triggers
- Comprehensive <a href="#">AppleScript and shell script</a> action execution
- Floating menu and <a href="#">HUD overlay</a> display for contextual shortcut access
- Cloud sync of <a href="#">settings and presets</a> across multiple Macs via iCloud
